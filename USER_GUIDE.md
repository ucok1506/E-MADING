## ğŸ’¡ Tips dan Troubleshooting

### Tips Penggunaan

**Untuk Siswa**:
1. **Gunakan judul menarik** dan deskriptif
2. **Pilih kategori tepat** agar mudah ditemukan
3. **Upload gambar berkualitas** untuk menarik pembaca
4. **Tulis konten informatif** dan bermanfaat
5. **Gunakan fitur draft** untuk artikel belum selesai
6. **Engage dengan komentar** pembaca

**Untuk Guru**:
1. **Review artikel cepat** (target 24 jam)
2. **Berikan feedback konstruktif** untuk siswa
3. **Maintain standar kualitas** yang konsisten
4. **Monitor progress siswa** secara berkala
5. **Share best practices** dengan guru lain

**Untuk Admin**:
1. **Backup database rutin** untuk keamanan data
2. **Monitor performa sistem** secara berkala
3. **Update keamanan** sistem secara rutin
4. **Analisis tren penggunaan** untuk improvement
5. **Kelola user accounts** secara proaktif

### Troubleshooting Umum

#### âŒ **Masalah Login**
- **Tidak bisa login**: Cek email/password, clear cache browser
- **Halaman loading terus**: Hard refresh (Ctrl+F5), cek koneksi
- **Error 403**: Logout-login ulang, hubungi admin

#### âŒ **Masalah Upload**
- **Gambar tidak upload**: Cek format (JPG/PNG/GIF), ukuran max 5MB
- **File dokumen gagal**: Format PDF/DOC/DOCX/PPT, max 10MB
- **Video tidak embed**: Gunakan embed URL, cek privacy settings

#### âŒ **Masalah Performa**
- **Loading lambat**: Clear cache, tutup tab tidak perlu, cek internet
- **Layout rusak**: Update browser, reset zoom ke 100%
- **Fitur tidak jalan**: Enable JavaScript, disable ad blocker

#### âŒ **Masalah Editor**
- **Text editor error**: Gunakan browser modern, enable JavaScript
- **Artikel tidak tersimpan**: Cek koneksi, login ulang jika session expired
- **Format hilang**: Hindari copy-paste langsung dari Word
